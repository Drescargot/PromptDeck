# 🧠 What is a prompt?

A prompt is simply the instruction you give to an AI.

It tells the AI:
- what role to take
- how to behave
- what goal to achieve

---

## 💬 Simple example

If you write:

> “Explain photosynthesis simply”

The AI will explain it.

But if you write a better prompt like:

> “You are a patient teacher explaining photosynthesis to a 12-year-old using simple examples”

👉 The answer becomes much clearer, more structured, and more useful.

---

## ⚡ Why prompts matter

AI is powerful, but it does NOT think like a human.

The quality of the result depends on:
> how clearly you guide it

Better prompt = better answer.

---

## 🧩 Types of prompts in this library

This repo uses 4 main styles:

### 🧠 Role prompts
You define a role:
> “You are a productivity coach…”

### 🎯 Goal prompts
You define an outcome:
> “Help me create a study plan…”

### 🧭 Behavior prompts
You define how the AI should act:
> “Be strict, structured, and direct…”

### 🔁 System prompts
Full structured instructions combining everything above.

---

## 🚀 Key idea

You are not “asking” AI.

You are **designing how it thinks and responds**.
